# Hari Haran - MERN Stack Portfolio

A fully dynamic personal portfolio website for an Information Technology student with complete admin panel.

## Features

- **User Side**: Home, About, Skills, Projects, Certificates, Testimonials with carousel, Contact form
- **Admin Panel**: Secure login, Dashboard with stats, CRUD operations for all content
- **Tech Stack**: React.js, Node.js, Express.js, MongoDB, JWT, Cloudinary
- **Design**: Dark theme, glassmorphism, responsive, animations

## Setup Instructions

### Prerequisites

- Node.js (v16+)
- MongoDB (local or Atlas)
- Cloudinary account (for image uploads)
